Homework 7 Object Model & Event Model

Part I

In Part I, the user must enter a number containing at least 4 decimal places. The number is then put through a series of calculations using Math.round, Math.floor, Math.sqrt, and toFixed. The results are then displayed to the user.

View Assignment Part I

Part II

In Part II, the user is prompted to enter a text of their choice. The user then enters a character to search for within their entered text. The JavaScript uses the charAt() string method to search for the desired character. The results are then outputed into a text box if found. If no character is matched within the text, a new window opens that announces the negative results to the user.
