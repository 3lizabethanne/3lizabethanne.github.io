CNIT 133 Assignments

JavaScript, jQuery, & AJAX

Homework 2 Basic Operations

Part I

In Part I, a simple script is created with various style for the different character couples. The script uses the document.write method to display characters. Inline CSS calls are made to add variations to the duo characters.

Part II

In Part II, an input form is used to gather user inputted numbers. Those numbers are then used in a series of calculations including sum, average, and product. The highest and lowest numbers are found using the Math.min and Math.max methods. The user is able to fade the results box with the use of a jQuery feature.

Part III

In Part III, an input form is displayed that calls for the user to enter 5 numbers. The numbers are then assessed based on criteria using if statements, and the results box displays the number of negative, positive, and zero integers received.
