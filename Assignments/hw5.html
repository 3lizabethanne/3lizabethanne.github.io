CNIT 133 Assignments

JavaScript, jQuery, & AJAX

Homework 5 Arrays

Part I

In Part I, a form is used for a travel survey, where the user must enter their name, check at least 1 country, select a language, and pick their preferred time of year to travel. Upon submitting the required criteria, the user is redirected to a submission page. If any of the required items are not filled out, a user error is issued and the form is not submitted.

Part II

In Part II, there are 2 pulldown menus. The 1st will take the user directly to their destination site when it is selected. The 2nd will require the user to click the "Go" button in order to take them to their destination.

Part III

Part III, uses a 2-dimensional array to collect user data of a state abbreviation or state name to then display statistics for the entered state. The user is allowed to enter in uppercase or lowercase. The function currently accepts US state names starting with letters A through C.
