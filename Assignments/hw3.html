CNIT 133 Assignments

JavaScript, jQuery, & AJAX

Homework 3 Functions

Part I

In Part I, a function is written that calculates students' final grades. First it takes user data, then it validates the entries, and lastly it displays results for final grades.

Part II

In Part II, a function calculates the weekly total sales and commission earnings for a single employee. The programs asks for a employee's name and then requires the total number of units sold for each priced item. The table displays the total for each item as well as grand total and then calculates the employee's earned comission.

Part III

In Part III, a single input box is displayed asking the user to enter a temperature. The user then chooses to convert the temperature to Celsius or Farenheit by clicking a button. The conversion results are then displayed for the user.

Extra Credit

In this Extra Credit, a program is created that tests the user's knowledge of mulitplication tables using the math random function. Their answer is evaluated and then feedback is giving letting them know if their input is correct or incorrect.
