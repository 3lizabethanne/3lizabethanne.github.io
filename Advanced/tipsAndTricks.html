Optimizing Images for the Web

For image optimization as a photographer I'm accustomed to using Adobe Photoshop, and I would highly recommened the program if you have access to it. However, for a quick non-software based alternative, the JPEG Optimizer is a straight-forward, easy to navigate internet site that will acheive the end goal in seconds.

The headshot image used on my About page was optimized using JPEG Optimizer. The starting file size was 56 KB and after compressing the file was down 65% to 19 KB!
